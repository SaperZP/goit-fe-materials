Консоль разработчика

При разработке скриптов всегда будут ошибки. Консоль показывает информацию, связанную с веб-страницей: сетевые запросы, JavaScript, CSS, ошибки безопасности и предупреждения, а также сообщения об ошибках, предупреждениях и сообщениях JavaScript, запущенным в контексте страницы. Позволяет вам взаимодействовать с веб-страницей, выполняя выражения JavaScript в контексте страницы.

Тег script

Для вставки скрипта в документ мы просто пишем <script>, без дополнительных атрибутов и комментариев

		<script>
			alert('Hello, World!');
		</script>

Внешние скрипты

Если JavaScript-кода много – его выносят в отдельный файл, который подключается в HTML.

		<script src="/path/to/script1.js"></script>
		<script src="/path/to/script2.js"></script>

Синхронные скрипты

Браузер загружает и отображает HTML постепенно. Если браузер видит тег <script>, то он по стандарту обязан сначала выполнить его, а потом показать оставшуюся часть страницы. Такое поведение называют «синхронным».

Асинхронные скрипты

Атрибут async

Скрипт выполняется полностью асинхронно. То есть, при обнаружении <script async src="..."> браузер не останавливает обработку страницы, а спокойно работает дальше. Когда скрипт будет загружен – он выполнится.

Атрибут defer

Скрипт также выполняется асинхронно, не заставляет ждать страницу, но есть два отличия от async.

		<script src="1.js" defer></script>
		<script src="2.js" defer></script>
		html markup
  • Браузер гарантирует, что относительный порядок скриптов с defer будет сохранён
  • Скрипт с defer сработает, когда весь HTML-документ будет обработан браузером

Переменные

  • Переменные используются для хранения информации.
  • Переменная состоит из имени и выделенной области памяти, которая ему соответствует.
  • Для объявления переменной используется ключевое слово var: var message;
  • После объявления, можно записать в переменную данные: message = 'Hello';
  • Эти данные будут доступны при обращении по имени: console.log( message );

Имена переменных

  • Имя может состоять из: букв, цифр, символов $ и _
  • Первый символ не должен быть цифрой
    • не верно - var 123myVariable;
    • верно - var myVariable123;
  • Регистр букв имеет значение, apple и aPplE две разные переменные
    • var apple;
    • var aPplE;
  • Зарезервированные имена var, class, return, export ...
  • Имя переменной должно быть понятным
    • плохо - var yabloko, chislo, moyaCar, naborIz4;
    • хорошо - var apple, num, myCar, list;

Типы данных

  • Число «number»: var n = 10;
  • Строка «string»: var str = "just some text";
  • Булевый тип «boolean»: var status = true;status = false;
  • Специальное значение «undefined»: var age = undefined;
  • Специальное значение «null»: var name = null;
  • Объекты «object»: var dog = { age: 5, name: "Rex" };

Основные операторы

Сложение, вычитание, умножение, деление, остаток от деления

				var a = 2 + 2; // 4
				var b = 3 - 2; // 1
				var c  = 3 * 3; // 9
				var d = 4 / 2; // 2
				var e = 5 % 2; // 1

Основные операторы

Конкатенация (сложение строк):

				var str = "My" + "String";
				console.log(str) //MyString

Приведение строки к числу:

				var a = 1, b = "2";
				a + Number(b) // 3
				a + +b // 3

Инкремент ++ и декремент --

Инкремент:

				var a = 2;
				a++;
				console.log(a); //3

Декремент:

				var b = 5;
				b--;
				console.log(b); //4

Операторы сравнения

  • Больше/меньше: a > b, a < b
  • Больше/меньше или равно: a >= b, a <= b
  • Равенство: a == b
  • Не равенство: a != b
  • Строгое равенство: a === b
  • Строгое неравенство: a !== b

Логические(Булевые) значения

  • true – имеет смысл «да», «верно», «истина», 1
  • false – означает «нет», «неверно», «ложь», 0
		console.log( 5 > 2 ); // true, верно
		console.log( 3 == 1 ); // false, неверно
		console.log( 3 != 1 ); // true, верно
		console.log( 0 == false ); // true, верно
		console.log( 0 === false ); // false, неверно

Взаимодействие с пользователем

  • alert: alert(text);
  • prompt: var name = prompt(text, default);
  • confirm: var isHuman = confirm(text);

Условный оператор if

Когда в зависимости от условия нужно выполнить различные действия, используется оператор if.

		var age = prompt("Введите ваш возраст:", '')
		if ( age < 18) {
			alert("младше 18");
		}

Неверное условие, else

Необязательный блок else («иначе») выполняется, если условие неверно.

		var age = prompt("Введите ваш возраст:", '')
		if (age < 18) {
			alert("младше 18");
		} else {
			alert("старше или ровно 18");
		}

Несколько условий, else if

Если нужно проверить несколько вариантов условия используется блок else if.

		var age = prompt("Введите ваш возраст:", '')
		if (age < 18) {
			alert("младше 18");
		} else if (age == 18) {
			alert("ровно 18");
		} else {
			alert("старше 18");
		}
